# Sistema de Gestión de Empleados - Construmara

## 📌 Descripción
Aplicación web desarrollada con Spring Boot para la gestión de empleados. Permite registrar, editar, eliminar y buscar empleados.

## 🚀 Tecnologías utilizadas
- Java 17
- Spring Boot
- Spring Data JPA
- Thymeleaf
- MySQL
- Maven

## ⚙️ Funcionalidades
- ✔ Registro de empleados
- ✔ Edición de empleados
- ✔ Eliminación con confirmación
- ✔ Búsqueda de empleados
- ✔ Mensajes de éxito y error

## 🧩 Estructura del proyecto
- Controller: Manejo de rutas
- Model: Entidad empleados
- Repository: Acceso a base de datos
- Templates: Vistas HTML

## 🗄️ Base de datos
- Nombre: empleados
- Motor: MySQL
